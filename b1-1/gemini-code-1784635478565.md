# [최종] 국어 학습 AI 시스템 개발 및 검증 보고서

---

## 1. 기능 동작 검증

### 1.1 업무 과업 정의
* **과업 명칭**: 중학생 대상 국어 문법 학습 콘텐츠 생성 및 맞춤형 질의응답 시스템
* **타겟 사용자**: 중학교 1~3학년 학생 (특히 문법 개념 파악이 어렵거나 내신을 대비하는 학습자)
* **입력 규격**: 학년, 단원, 세부 주제, 학습자 수준, 원하는 결과, 문제 형식, 난이도, 톤, 주의사항 (총 9개 필드)
* **출력 규격**: 선택한 난이도/형식에 맞춘 개념 설명, 예문, 평가 문항, 정답 및 오답 해설 (불릿 형태, 내부 추론 과정 제외)

---

### 1.2 입력 템플릿 (재사용 가능)
```text
[국어 학습 과업 입력 템플릿]

- 학년: (예: 중3)
- 단원: (예: 문장 성분)
- 세부 주제: (예: 주어, 서술어, 목적어, 보어 구분)
- 학습자 수준: (예: 기초 / 중간 / 상위권)
- 원하는 결과: (예: 개념 설명 / 문제 생성 / 오답 정리 / 서술형 답안 첨삭)
- 문제 형식: (예: 객관식 / 단답형 / 서술형 / 혼합형)
- 난이도: (예: 하 / 중 / 상)
- 톤: (예: 친절한 중등 국어 선생님 말투)
- 주의사항: (예: 어려운 용어는 쉽게 풀어서 설명하고, 모르는 규정은 단정하지 말 것)
